# PQViewer (Line)

[English](README.md) | 日本語

ブラウザ上で動作する軽量な2Dプロットビューアです。CSV/TSVなどのデータを読み込み、インタラクティブに可視化できます。

---

## 主な機能

- CSV / TSV / 空白区切りデータのドラッグ&ドロップ読み込み
- 複数系列の描画（線・点・エラーバー）
- 軸設定（範囲、対数スケール、tick形式など）
- 凡例・ラベル・フォントの詳細設定
- マルチプロット（subplot）対応
- PNGエクスポート
- 設定のJSONエクスポート/インポート
- 簡易的なgnuplot風コマンド入力
- アニメーション（フレーム再生）

---

## 使い方

1. [PQViewer (Line)](https://mfukudaqed.github.io/PQViewer_Line/)ページを開く
2. データファイル（CSV/TSVなど）をドラッグ＆ドロップ
3. Control Panelで各種設定を調整
4. 必要に応じてPNGとして保存

データ形式:
- 1列目: x
- 2列目以降: y系列
